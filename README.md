## Building

- Clone the repository.
  ```bash
  git clone https://github.com/jainil63/Jarvis.git
  ```

- Make a env file
  ```bash
  CHROME_PATH = 
  ```

- For run the app.
  ```bash
  uv run python main.py
  ```

## Developer Commands

- For formatting code, we use `ruff`
  ```bash
  uv run ruff format .
  ```
